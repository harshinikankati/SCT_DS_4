# Traffic Accident Analysis – UK Dataset 🚦

## 📌 Project Overview
This project focuses on analyzing traffic accident data from the United Kingdom to identify key patterns related to **road conditions, weather, and time of day**. The analysis aims to uncover accident hotspots and major contributing factors using interactive visualizations built in **Power BI**.

## 🎯 Objectives
- Identify accident patterns based on:
  - Road surface conditions
  - Weather conditions
  - Time of day and day of week
- Visualize accident hotspots across locations
- Highlight key contributing factors influencing accident severity
- Present insights through clear and interactive dashboards

## 📊 Tools Used
- Power BI  
- DAX (for calculated columns and measures)

## 📈 Dashboards Created
### 1️⃣ Accident Analysis Dashboard
- Total accidents (KPI Cards)
- Accidents by road surface condition
- Accidents by weather condition
- Accidents by day of week

### 2️⃣ Time & Location Analysis Dashboard
- Accidents by time of day
- Accident hotspots using map visualization
- Severity distribution across different conditions

## 🔍 Key Insights
- Higher number of accidents occurred on **dry roads**, indicating volume-driven risk.
- Poor weather conditions such as **rain and fog** showed increased accident severity.
- Accident frequency peaked during **morning and evening rush hours**.
- Weekdays recorded more accidents compared to weekends.
- Urban areas showed dense accident hotspots.

## 📂 Dataset
- UK Road Safety / Traffic Accident Dataset

## ✅ Conclusion
This analysis provides valuable insights into how environmental and temporal factors affect road safety. The findings can help policymakers and traffic authorities improve road infrastructure and accident prevention strategies.
